4. О языке Java

Java используют 3 миллиона устройств.

Этот язык универсален и используется повсюду: мобильные приложения и игры, научные разработки и мобильные телефоны, игровые консоли и серьёзные суперкомпьютеры.

Пожалуй, проще сказать, кто в интернете не использует Java. На этом языке написаны amazon.com — популярнейший интернет-магазин; серверная часть ebay, весь набор функций операционной системы Android, и даже спецэффекты в фильме «Мстители» компания созданы с помощью Java.
Язык Java популярен и в России. С помощью него создана серверная часть сайта РЖД и соцсеть Одноклассники. На Java также пишут корпоративные приложения для создания программного обеспечения, такие как JetBrains IntelliJ Idea, Jira, NetBeans, RubyMine, Eclipse и другие.
